汴地吃货 API 文档

API 版本：1

GET*, POST*: need login
data_type: required parameter
data_type: optional parameter

数据类型

  • bool 1
  • int 31415926
  • str "this is a string"
  • time "2012-07-14 00:00:00"
  • date "2012-07-14"

HTTP 状态码

  • 200 正常。
  • 400 请求格式错误。
  • 403 没有权限。（如未登录）
  • 404 请求的资源不存在。
  • 501 尚未实现此功能。
  • 503 暂时无法提供服务。
  • 418 我是一只咖啡壶☆

Error Code

若返回的字典含有 error_code，客户端要处理错误，且不应将 error_code 本身暴露给用户。

若同时含有 error，要将 error 的内容显示给用户。

如遇 NotLoggedIn，应将用户跳转至登录界面。
